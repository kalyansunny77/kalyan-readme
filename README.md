## 🚀 Featured Project

### 🛒 [E-Commerce Microservices Platform](https://github.com/kalyansunny77/ecommerce-microservices)

> Production-grade event-driven microservices system — the kind of architecture powering real enterprise applications

```
Customer places order
        │
        ▼
  [Order Service] ──── publishes ────► [Kafka]
  Java 21 + Spring Boot                   │
  PostgreSQL + JWT Auth          ┌─────────┴──────────┐
                                 ▼                     ▼
                        [Inventory Service]   [Notification Service]
                         MongoDB + Kafka       MongoDB + Kafka
                         checks stock          sends confirmation
```

**Tech Stack:**
`Java 21` `Spring Boot 3` `Apache Kafka` `PostgreSQL` `MongoDB` `Docker` `Kubernetes` `JWT` `Flyway` `Swagger`

**What it demonstrates:**
- ✅ Event-driven architecture — services communicate only through Kafka, fully decoupled
- ✅ JWT-secured REST APIs with 7 endpoints and full Swagger documentation
- ✅ Flyway database migrations for versioned, repeatable schema changes
- ✅ 85%+ unit test coverage with JUnit and Mockito
- ✅ Docker Compose for local orchestration of all services
